# Cinema Booking App

A simple cinema booking application built with React, Node.js, and Docker.

This repository contains a small movie booking frontend and backend setup that can be started with Docker Compose.

## Features

- Browse movies and showtimes
- View available seats
- Book tickets through a simple checkout flow
- Basic service structure for learning a full-stack application

## Tech Stack

- React for the frontend
- Node.js for backend APIs
- Docker Compose for local development

## Quick Start

```powershell
cd d:\cinema

docker compose up -d --build
```

Open the app in your browser at `http://localhost`.
